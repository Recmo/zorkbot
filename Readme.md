# Zork Slackbot

Requires

[ ] Create a bot account on Slack, store the bot api token in a file called `API_TOKEN`.
[ ] Install `frotz`.
[ ] Download `ZORK1.DAT` and store in the folder.
[ ] Start with `yarn run run`
[ ] Get channel id from log
[ ] Set channel id in source file
[ ] Restart
